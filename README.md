# 🌐 SmartName — AI-Powered Domain Name Recommendation System

**SmartName** is an intelligent domain name recommendation system powered by AI and Natural Language Processing (NLP). It suggests creative and relevant domain names based on user input. Ideal for entrepreneurs, developers, and branding teams launching new digital products or businesses.

---

## 🚀 Features

- 🔍 **Context-Aware Suggestions**: Generates names tailored to user-provided keywords or descriptions.
- 🌍 **Language Support**: Suggests domain names in English, Sinhala, or a mix.
- 🧠 **AI-Powered Creativity**: Uses fine-tuned large language models (LLMs) to generate relevant, brandable domains.
- 📦 **Vector Search Engine**: Retrieves similar concepts using FAISS for semantic similarity.
- 🌐 **Frontend + Backend**: User-friendly React interface with a FastAPI backend.
